# myComponet
使用dumi搭建的组件库，并尝试了部署在nginx上面
## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father`,

```bash
$ npm run build
```
生成部署文件
```
npm rundocs:build
```
